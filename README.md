#Mufasa#

Mufasa is a Annotation HTTP Client library for Java

## Installation ##

Direct Download (2016/11/21)

The lastest stable is mufasa-1.0.tar.gz (release notes)

Maven

	<dependency>
	    <groupId>org.pinae</groupId>
	    <artifactId>mufasa</artifactId>
	    <version>1.0</version>
	</dependency>
